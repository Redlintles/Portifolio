Descrição

* O Curso já está atualizado com a versão 2 do Tailwind (jan/2021)

Tailwind foi criado para revolucionar o desenvolvimento de aplicações web, a abordagem utility first agiliza muito o desenvolvimento, pois gastamos menos tempo com CSS e conseguimos construir layouts incríveis com poucas linhas de código.

Neste curso você vai aprender Tailwind CSS do básico ao avançado, todos os recursos do framework são amplamente explicados com o auxílio de exemplos reais, simulando situações que ocorrem no dia a dia de um programador.

O curso conta ainda com exercícios que estimularão o seu aprendizado, para reforçar seu conhecimento sobre os tópicos e seções teóricas, além de desafiar você a atingir o próximo nível e dominar a ferramenta.

E ainda construiremos juntos projetos incríveis baseados em aplicações reais, desde a primeira linha de código até o projeto pronto para a produção, utilizando tudo que foi abordado no curso e ainda Técnicas Modernas como: mobile first, HTML semântico, estruturação de projeto e muito mais!

Veja alguns dos recursos abordados no curso:

  • Instalação do Tailwind;

  • Customização do framework;

  • Layout mobile first;

  • Construção de layout com Grid e Flexbox;

  • Criação de projetos web;

  • Princípios de arquitetura de software;

  • Boas práticas e clean code;

  • Spacing, sizing, tipografia, background, borders e muito mais!

    •

O foco principal do curso é lhe ensinar a como utilizar a ferramenta de forma profissional, explorando todas as funcionalidades de Tailwind e situações que acontecem no desenvolvimento de projetos, para que você se prepare para utilizar de forma profissional.

Ao finalizar o curso você estará apto a utilizar o Tailwind de forma avançada, criando qualquer layout que for preciso ou replicar qualquer site que você deseje.

O curso é voltado para a prática, este é o último curso que você vai precisar fazer para aprender e dominar o Tailwind.

Está pronto para criar layouts incríveis de uma forma fácil e simples?

Te espero então para desvendarmos o Tailwind CSS juntos! :)

O que você aprenderá:

  • Básico ao avançado de Tailwind CSS

  • Criar projetos de Tailwind com Grid CSS e Flexbox

  • Criar layouts modernos com Tailwind CSS

  • Responsividade com Tailwind

  • Desenvolvimento Mobile First

Há algum requisito ou pré-requisito para o curso?
